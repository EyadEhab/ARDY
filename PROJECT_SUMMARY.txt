PROJECT OVERVIEW
================================================================================

EgyptAgri-Pulse is a comprehensive Digital Twin of the Egyptian agricultural
landscape that bridges real-time environmental intelligence, biochemical soil
science, and macro-economic historical trends to provide localized, actionable
roadmaps for farmers and policymakers.

================================================================================
ARCHITECTURE (4-LAYER SYSTEM)
================================================================================

Layer 1: Live Environmental Telemetry (Real-Time)
  ✓ OpenWeatherMap API integration
  ✓ Real-time weather fetching for 22 Egyptian governorates
  ✓ Temperature, humidity, rainfall data
  ✓ Fallback to synthetic data for demo purposes

Layer 2: Precision Crop Recommendation (Classification)
  ✓ Ensemble ML models: XGBoost + Random Forest
  ✓ Input: N, P, K levels and pH values
  ✓ Output: Recommended crop from 22 species
  ✓ Accuracy: 64.5% on test set
  ✓ Confidence scores and top-5 alternatives

Layer 3: National Yield Forecasting (Regression)
  ✓ Time-series analysis of 34 years of FAO data (1990-2024)
  ✓ Individual models for 10 major Egyptian crops
  ✓ 2026 yield predictions
  ✓ R² scores: 0.44 to 0.78 across crops

Layer 4: Intelligent Dashboard (Deployment)
  ✓ Streamlit-based interactive dashboard
  ✓ Geospatial mapping with Folium
  ✓ Feature importance analysis
  ✓ PDF report generation
  ✓ Real-time visualization with Plotly

================================================================================
API ENDPOINTS
================================================================================

Health & Reference:
  GET  /api/health                    - System health check
  GET  /api/governorates              - List all governorates

Weather & Environment:
  GET  /api/weather/<governorate>     - Get live weather data

Crop Recommendation:
  POST /api/recommend-crop            - Get crop recommendation
       Input: {n, p, k, ph}
       Output: {crop, confidence, top_5_crops}

Yield Forecasting:
  POST /api/forecast-yield            - Forecast crop yield
       Input: {crop, year}
       Output: {predicted_yield, historical_avg, r2_score}

Explainability:
  POST /api/shap-explanation          - Get feature importance
       Input: {n, p, k, ph}
       Output: {base_value, feature_importance}

Reporting:
  POST /api/generate-report           - Generate PDF report
       Input: {governorate, n, p, k, ph}
       Output: PDF file
