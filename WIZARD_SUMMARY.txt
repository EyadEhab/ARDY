STEP 2: ENTER SOIL DATA & GET RECOMMENDATIONS (🧪 Soil Input)
────────────────────────────────────────────────────────────
User Actions:
  • Adjust sliders for N, P, K, pH
  • View current values in table
  • Click "Get Crop Recommendations"

Data Collected:
  • Nitrogen (N): 0-140 mg/kg
  • Phosphorus (P): 5-145 mg/kg
  • Potassium (K): 5-205 mg/kg
  • Soil pH: 3.5-10.0

Processing:
  • Combine soil data + weather data
  • Run through XGBoost model
  • Run through Random Forest model
  • Average probabilities (ensemble voting)
  • Generate explanations for top 3 crops

Output:
  • Top 3 recommended crops
  • Confidence scores (0-100%)
  • Detailed explanations (expandable)
  • Model comparison (XGBoost vs Random Forest)

  ✓ Soil pH (6.5) is optimal for Cotton
